# OrderDeliveryMvcApp

Для запуска проекта необходимо:
- В файле appsetttings.json прописать собственную строку подключения к базе данных
- В консоли диспетчера пакетов Nuget прописать команду для обновления базы данных до последней миграции (для Rider - dotnet ef database update --project OrderDeliveryWebApplication, для VisualStudio – Update-Database)
